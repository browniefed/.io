Jama Node API - Released/Active Development

Jama Node - Github

This is a simple REST API caller written in JavaScript. Don't let it being called Jama Node fool you, this can be used on the front end as long as you are running Jama REST Passthrough to allow for CORS. The Jama REST API can be utilized using NPM

npm install jama-rest-api --save

Jama REST Passthrough - Released/Active Development

Jama Passthrough - Github

This is a REST Passthrough that allows for CORS (cross origin resource sharing) with the Jama REST API. The Jama Passthrough can be utilized using NPM

npm install jama-passthrough --save 

var JamaPassthrough = require('jama-passthrough');
var restEndpoint = 'http://localhost:8080/contour/rest/v1'; // YOUR REST API HERE
var optionalCORS = ['*']; //* for development but in production I recommend limiting to your app host

var JamaAPI = new JamaPassthrough({
    restEndpoint: restEndpoint,
    allowedCORS: optionalCORS,
    port: 9999
});
JamaAPI.start();//Listens on Port 9999 unless otherwise specified

Jama Universal App - In Development

Jama Universal - Github

This is a universal web application for Jama. It can be packaged up 3 different ways. It can compiled to a mobile app using phonegap, a desktop app using node-webkit, or a standalone web application.
