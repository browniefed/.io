Jama Node API - Released

Jama Node - Github

This is a simple REST API caller written in JavaScript. Don't let it being called Jama Node fool you, this can be used on the front end as long as you are running Jama REST Passthrough to allow for CORS.

Jama REST Passthrough - Released

Jama Passthrough - Github

This is a REST Passthrough that allows for CORS (cross origin resource sharing) with the Jama REST API.

Jama Universal App - In Development

Jama Universal - Github

This is a universal web application for Jama. It can be packaged up 3 different ways. It can compiled to a mobile app using phonegap, a desktop app using node-webkit, or a standalone web application.
